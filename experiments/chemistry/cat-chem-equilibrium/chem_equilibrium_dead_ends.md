# Chemical Equilibrium — Dead Ends Log

## DEAD END 1 — Treating Kp = Kc
**What was attempted**: Using Kp directly as Kc without delta_n correction.
**Result**: Off by factor of (RT)^delta_n for Haber-Bosch.
**Why this is a dead end**: Kp = Kc * (RT)^Δn; when Δn ≠ 0, they differ significantly.
**Do NOT repeat**: Assuming Kp equals Kc.

## DEAD END 2 — Positive sign in van't Hoff
**What was attempted**: ln(K2/K1) = +(ΔH/R)(1/T2 - 1/T1).
**Result**: Predicts K increases with T for exothermic reactions (wrong).
**Why this is a dead end**: Correct form has MINUS sign.
**Do NOT repeat**: Using positive sign in van't Hoff equation.
